# Présentation du projet
  Rédaction d'un script python pour la detection du visage, des personnes et de leurs émotions
  au travers d'un flux caméra. 
  A titre d'exemple j'utilise un database de célébrité disponible sur kaggle :
      - 

  ## Etapes

    - Reconnaissances du Visage detecter (OpenFace) ⏳
    - Reconnaissance des Emotions (FER) ⏳
